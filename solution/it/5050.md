# 5050 【B-325-E】最快到达

## 题目描述

某国有 N 座城市。

您将从位于城市 1 的办公室出发，途经零个或多个城市到达位于城市 N 的目的地。

有两种交通工具可供选择：汽车和火车。从城市 i 到城市 j 所需的时间如下、

- 乘坐汽车$D_{i,j} \times$A 分钟。
- 乘坐火车$D_{i,j} \times$B+C 分钟。

您可以从汽车换乘火车，但不能从火车换乘汽车。
您可以不花时间换乘，但只能在城市中换乘。

从城市 1 到城市 N 最短需要多少分钟？

注意：

- 2$\le$ N$\le$1000

- 1$\le$A,B,C$\le$10^6

- $D_{i,j} \le$10^6

- $D_{i,j}$= 0

- $D_{i,j}$=$D_{i,j} >$0 (i$\neq$j)

    所有输入值均为整数。

## 输入

N A B C

${D}_{1,1}\,{D}_{1,2}\,...\,{D}_{1,N}$

${D}_{2,1}\,{D}_{2,2}\,...\,{D}_{2,N}$

.

.

.

${D}_{N,1}\,{D}_{N,2}\,...\,{D}_{N,N}$

## 提示/说明

样例一：

通过以下移动方式，您可以在 78 分钟内从城市 1 到达城市 4 。

- 乘坐汽车从城市 1 到城市 3 。耗时 2×8=16 分钟。
- 乘坐汽车从城市 3 前往城市 2 。耗时 3×8=24 分钟。
- 乘坐火车从城市 2 前往城市 4 。耗时 5×5+13=38 分钟。

从城市 1 到城市 4 不可能少于 78 分钟。

## 解题思路

![image-20260406135017917](.\image-huancheng.png)

### 1.存图方式改变

1.e边数组：**不需要**，用head[N],to[M],nxt[M],w[M]

- **head**每个顶点第一条边的编号

- **to**每条边的结束

- **nxt**下一条边的编号

- **w**权值

2.**建图**：

邻接矩阵：

```c++
e[i][i+n]=0;
e[i][j]=x*a;
e[i+n][j+n]=b*x+c; 
```

链式前向星：

```c++
add(i,i+n,0);
add(i,j,x*a);
add(i+n,j+n,b*x+c);
```

**邻接矩阵**是**直接赋值**，而**链式前向星**是使用**函数添加**

### 2.松弛（**relax函数**）参数对比：

 ` relax(t.id,i,e[t.id][i]) `对比 `relax(t.id,v,h)`，只是来源不同，数值是一样的

### 3.dijkstra函数对比

**邻接矩阵**：

```c++
void dijk(){
	init();
	f[1]=0;
	q.push({1,0});
	while(!q.empty()){
		node t=q.top();
		q.pop();
		if(vis[t.id]) continue;
		vis[t.id]=1;
		for(int i=1;i<=(n<<1);i++){
			if(vis[i]<0x3f3f3f3f3f3f3f3f&&relax(t.id,i,e[t.id][i])){
				q.push({i,f[i]});
			}
		}
	}
}
```

**链式前向星**：

```c++
void dijk(){
	init();
	f[1]=0;
	q.push({1,0});
	while(!q.empty()){
		node t=q.top();
		q.pop();
		if(vis[t.id]) continue;
		vis[t.id]=1;
		for(int i=head[t.id];i;i=nxt[i]){
            int v=to[i];
            int h=w[i];
            if(relax(t.id,v,h)) q.push({v,f[v]});
        }
	}
}
```

`init()`函数：没有任何区别

`f[1]=0`：将第一个点设置为未访问，相同

`q.push({1,0})`：将第一个点和距离推进去，因为本身就在这个点，所以距离是0

`while`循环：不断进行松弛

​	`node t=q.top()`取出队列中距离最小的顶点（堆顶元素），赋值给 `t`

​	`q.pop()`弹出

​	`if(vis[t.id]) continue;`如果这个点已经确认为最短路，那么跳过

​	`vis[t.id]=1;`确认最短路

​	`for(int i=head[t.id];i;i=nxt[i])` ：

   - `head[t.id]`：第一条边的编号
   - `i=nxt[i]`：沿着链表走到下一条边
   - `i` 为 0 时结束（0 表示空指针）

​	`if(relax(t.id,v,h)) q.push({v,f[v]});`尝试松弛

​	调用 `relax` 函数尝试松弛：

- 如果 `f[t.id]+h<f[v]`，说明找到了一条更短的路径
- 更新 `f[v]` 为更小的值，返回 `1`
- 如果更新成功，将顶点 `v` 及其新的最短距离 `f[v]` 加入优先队列（以便后续用 `v` 继续松弛其他点）

### 输出

```c++
cout<<min(f[n],f[2*n])<<endl;
```

`f[n]`表示一直乘坐汽车

`f[2*n]`表示在**中途某个时刻**换乘火车

输出最小值，结束

## 参考代码

邻接矩阵：

```c++
#include <bits/stdc++.h>
#define int long long
#define N 2005
using namespace std;
int n,a,b,c;
int e[N][N];
int f[N];
void init(){
	for(int i=1;i<=(n<<1);i++){
		f[i]=0x3f3f3f3f3f3f3f3f;
	}
}
bool relax(int fr,int to,int w){
	if(f[to]>f[fr]+w){
		f[to]=f[fr]+w;
		return 1;
	}
	return 0;
}
struct node{
	int id;
	int w;
	bool operator<(const node&a)const{
		return w>a.w;
	}
};
bool vis[N];
priority_queue<node>q;
void dijk(){
	init();
	f[1]=0;
	q.push({1,0});
	while(!q.empty()){
		node t=q.top();
		q.pop();
		if(vis[t.id]) continue;
		vis[t.id]=1;
		for(int i=1;i<=(n<<1);i++){
			if(vis[i]<0x3f3f3f3f3f3f3f3f&&relax(t.id,i,e[t.id][i])){
				q.push({i,f[i]});
			}
		}
	}
}
signed main(){
	ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
	cin>>n>>a>>b>>c;
	memset(e,0x3f,sizeof(e));
	for(int i=1;i<=n;i++){
		for(int j=1;j<=n;j++){
			int x;
			cin>>x;
			e[i][i+n]=0;
			e[i][j]=x*a;
			e[i+n][j+n]=b*x+c; 
		}
	}
	memset(f,0,sizeof(f));
	dijk();
	cout<<min(f[n],f[2*n])<<endl;
	return 0;
}
```

链式前向星：

```c++
#include <bits/stdc++.h>
#define int long long
#define N 2005
#define M 4000005 
using namespace std;
int n,a,b,c;
int head[N],to[M],nxt[M],w[M],cnt;
int f[N];
bool vis[N];
void init(){
	for(int i=1;i<=(n<<1);i++){
		f[i]=0x3f3f3f3f3f3f3f3f;
		vis[i]=0;
	}
}
bool relax(int fr,int to,int w){
	if(f[to]>f[fr]+w){
		f[to]=f[fr]+w;
		return 1;
	}
	return 0;
}
struct node{
	int id;
	int w;
	bool operator<(const node&a)const{
		return w>a.w;
	}
};
priority_queue<node>q;
void add(int u,int v,int h){
	to[++cnt]=v;
	w[cnt]=h;
	nxt[cnt]=head[u];
	head[u]=cnt;
}
void dijk(){
	init();
	f[1]=0;
	q.push({1,0});
	while(!q.empty()){
		node t=q.top();
		q.pop();
		if(vis[t.id]) continue;
		vis[t.id]=1;
		for(int i=head[t.id];i;i=nxt[i]){
            int v=to[i];
            int h=w[i];
            if(relax(t.id,v,h)) q.push({v,f[v]});
        }
	}
}
signed main(){
	ios::sync_with_stdio(0),cin.tie(0),cout.tie(0);
	cin>>n>>a>>b>>c;
	memset(head,0,sizeof(head));
	for(int i=1;i<=n;i++){
		for(int j=1;j<=n;j++){
			int x;
			cin>>x;
			add(i,i+n,0);
            add(i,j,x*a);
            add(i+n,j+n,b*x+c);
		}
	}
	memset(f,0,sizeof(f));
	dijk();
	cout<<min(f[n],f[2*n])<<endl;
	return 0;
}
```
