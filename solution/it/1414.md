# 1414 家谱题目题解

## 1.定义结构体

我们把结构体数组中`name`数组的**第一项**作为父亲（上一代），后面**全部作为孩子**，每次访问第一个就相当于是他的父亲，这样可以**快速**、**便捷**的找到某个人的父亲。

但是，**每个人**的生的孩子的**数量**（`name`数组的**长度**）不同，查找时**需要孩子数量**，我们在**结构体**中定义一个`cnt`变量，作为他的孩子个数。

结构体代码：

```c++
struct person{
	string name[15];
	int cnt;
}p[50005];
```

## 2.输入输出

这道题的输入输出不太一样，使用符号进行的操作，所以在输入是要**截取字串**，要截取下**符号后面的名字**。

1. 放一个`while(1)`死循环，然后输入字符串`s`，如果是`$`先存放`cnt`（每个人的孩子数量）再`break`跳出循环
2. 如果**不是**`?`（查询符号），分两种情况：
    - 创建新家族（`#`符号）：存放`cnt`（每个人的孩子数量），再把长度变量`len++`，再把`x`（每个人的孩子的第几个）设为一，最后把`name`数组的第一个放成这个人的名字
    - 加入家族（`+`符号）：存放前`++x`，再存放`s`。
3. 如果是`?`（查询操作）：先存放`cnt`，再输出自己的名字，接着空格，然后输出祖先的名字，最后换行`cout<<s.substr(1)<<" "<<find(s.substr(1))<<endl;`

为什么要截取`s.substr(1)`？

因为第一个是**操作符号**，不需要，直接跳过，调用时直接调用`s.substr(1)`（从第一个位置开始截取字符串到结尾）

代码：

```c++
string s;
int x=0;
while(1){
	cin>>s;
	if(s=="$"){
		p[len].cnt=x;
		break;
	}
	if(s[0]=='#'){
		p[len].cnt=x;
		len++,x=0;
		p[len].name[++x]=s.substr(1);
	}else if(s[0]=='+') p[len].name[++x]=s.substr(1);
	else if(s[0]=='?'){
		p[len].cnt=x;
		cout<<s.substr(1)<<" "<<find(s.substr(1))<<endl;
	}
}
```

## 3.查找祖先函数

我采用的是递归的函数的查找，这样可以很方便的查找出某个人的祖先：

1. 先在里面查找他自己在哪里
2. 返回调用函数查找上一代
3. 如果找不到上一代，说明自己就是最上层祖先，返回自己

查找代码：

```c++
string find(string s){
	for(int i=1;i<=len;i++){
		for(int j=2;j<=p[i].cnt;j++){
			if(s==p[i].name[j]) return find(p[i].name[1]);
		}
	}
	return s;
}
```

## 4.最终完整代码

```c++
#include <bits/stdc++.h>
using namespace std;
int len=0;
struct person{
	string name[15];
	int cnt;
}p[50005];
string find(string s){
	for(int i=1;i<=len;i++){
		for(int j=2;j<=p[i].cnt;j++){
			if(s==p[i].name[j]) return find(p[i].name[1]);
		}
	}
	return s;
}
int main(){
	string s;
	int x=0;
	while(1){
		cin>>s;
		if(s=="$"){
			p[len].cnt=x;
			break;
		}
		if(s[0]=='#'){
			p[len].cnt=x;
			len++,x=0;
			p[len].name[++x]=s.substr(1);
		}else if(s[0]=='+') p[len].name[++x]=s.substr(1);
		else if(s[0]=='?'){
			p[len].cnt=x;
			cout<<s.substr(1)<<" "<<find(s.substr(1))<<endl;
		}
	}
	return 0;
}
```
